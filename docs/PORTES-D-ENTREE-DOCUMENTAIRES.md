# PORTES D'ENTRÉE DOCUMENTAIRES

## Objet

Ce document définit les différentes portes d'entrée permettant d'explorer le patrimoine documentaire d'InsouWiki.

Une porte d'entrée documentaire constitue le premier point de contact entre une intention du lecteur et le patrimoine documentaire.

Elle ne détermine jamais le parcours d'exploration.

Elle en constitue simplement le point de départ.

---

# Principe fondamental

Une exploration documentaire peut commencer de nombreuses façons.

Tous les lecteurs n'abordent pas un sujet avec la même intention.

Certains connaissent déjà un auteur.

D'autres disposent d'une citation.

D'autres encore souhaitent comprendre un thème ou vérifier une affirmation.

InsouWiki reconnaît cette diversité et propose plusieurs portes d'entrée complémentaires.

---

# Les principales portes d'entrée

## Un auteur

Le lecteur souhaite explorer le parcours documentaire d'une personne.

Cette exploration peut conduire vers :

* les sujets documentaires abordés ;
* les observations documentaires ;
* les documents associés ;
* les preuves disponibles.

---

## Un sujet documentaire

Le lecteur souhaite comprendre un thème particulier.

Le sujet documentaire devient alors le centre de l'exploration.

Il relie progressivement :

* les observations ;
* les connaissances ;
* les faits ;
* les documents.

---

## Un document

Le lecteur commence par un document précis.

Il peut ensuite découvrir :

* les sujets abordés ;
* les faits documentaires extraits ;
* les observations auxquelles ce document contribue.

Le document devient alors une porte d'entrée vers un ensemble documentaire plus vaste.

---

## Une citation

Le lecteur connaît une phrase ou une affirmation.

InsouWiki permet de retrouver :

* son contexte documentaire ;
* les séquences concernées ;
* le document d'origine ;
* les autres observations associées.

La citation devient ainsi un point de départ vers l'ensemble du patrimoine documentaire.

---

## Une question

Le lecteur formule une question.

Cette question est progressivement rapprochée des sujets documentaires concernés.

L'objectif n'est pas de produire immédiatement une réponse.

L'objectif est d'orienter le lecteur vers le patrimoine documentaire pertinent.

---

# Futures portes d'entrée

L'architecture d'InsouWiki permet d'accueillir naturellement d'autres portes d'entrée, notamment :

* une période ;
* une date ;
* une organisation ;
* un lieu ;
* un événement documentaire ;
* un document récemment ajouté.

Ces nouvelles portes d'entrée enrichissent l'exploration sans modifier les principes fondamentaux du projet.

---

# Une même destination

Quelle que soit la porte d'entrée choisie, le lecteur rejoint progressivement le même patrimoine documentaire.

Toutes les portes d'entrée convergent vers :

* les sujets documentaires ;
* les observations documentaires ;
* les preuves ;
* les documents.

Il n'existe donc pas plusieurs patrimoines documentaires.

Il existe plusieurs façons d'explorer le même patrimoine.

---

# Le rôle de l'intelligence artificielle

L'intelligence artificielle peut aider le lecteur à identifier la porte d'entrée la plus pertinente.

Elle peut également suggérer d'autres parcours documentaires susceptibles d'enrichir l'exploration.

Ces suggestions demeurent toujours fondées sur le patrimoine documentaire disponible.

Elles ne remplacent jamais les choix du lecteur.

---

# Vision

Une bonne bibliothèque ne possède pas une seule entrée.

Elle permet à chacun de commencer son exploration là où se trouve sa curiosité.

InsouWiki adopte cette même philosophie.

Chaque porte d'entrée respecte une intention différente.

Toutes conduisent progressivement vers un patrimoine documentaire commun, organisé autour des preuves, des observations et des documents.

L'exploration documentaire n'impose jamais un parcours unique.

Elle accompagne le lecteur dans la construction de sa propre compréhension.
