# DOMAINE — RAISONNEMENT DOCUMENTAIRE

## Objet

Ce document définit la notion de raisonnement documentaire au sein d'InsouWiki.

Il ne décrit pas un objet métier particulier.

Il décrit la manière dont InsouWiki produit des observations documentaires à partir du patrimoine documentaire.

---

# Définition

Un raisonnement documentaire est un processus qui transforme un ensemble de preuves documentaires en une observation documentaire vérifiable.

Le raisonnement ne crée jamais de nouvelles preuves.

Il organise, rapproche ou interprète uniquement des preuves déjà présentes dans le patrimoine documentaire.

---

# Les preuves précèdent toujours le raisonnement

Les preuves documentaires constituent le fondement de tout raisonnement.

Un raisonnement ne peut utiliser que des éléments documentaires déjà établis :

* faits documentaires ;
* relations documentaires ;
* connaissances documentaires ;
* sujets documentaires.

Aucun raisonnement ne peut remplacer ou modifier ces éléments.

---

# Le raisonnement précède l'observation

Une observation documentaire n'est jamais produite directement.

Elle résulte d'un raisonnement explicite.

Le lecteur doit pouvoir comprendre comment cette observation a été construite.

---

# Le lecteur peut toujours vérifier

Tout raisonnement documentaire doit permettre au lecteur de revenir aux preuves qui le fondent.

La traçabilité documentaire constitue une propriété essentielle du raisonnement.

---

# Principe de prudence

Lorsque les preuves sont insuffisantes ou ambiguës, le raisonnement documentaire s'abstient de conclure.

L'absence de conclusion constitue un résultat documentaire légitime.

---

# Les familles de raisonnements

Les raisonnements documentaires pourront progressivement couvrir différents besoins :

* identification d'évolutions ;
* comparaisons documentaires ;
* chronologies ;
* convergences ;
* divergences ;
* parcours documentaires.

Toutes ces familles partagent les mêmes principes méthodologiques.

---

# Vision

Le rôle d'InsouWiki n'est pas de répondre à la place du lecteur.

Son rôle est de rendre les raisonnements documentaires suffisamment transparents pour que le lecteur puisse exercer son propre jugement à partir des preuves disponibles.
