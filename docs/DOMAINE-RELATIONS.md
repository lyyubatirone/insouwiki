# DOMAINE — RELATIONS DOCUMENTAIRES

## Objet

Ce document définit la notion de relation documentaire dans InsouWiki.

Il précise la manière dont plusieurs faits documentaires peuvent être reliés afin de construire des connaissances documentaires vérifiables.

---

# Principe fondamental

Une relation documentaire décrit un lien observable entre plusieurs faits documentaires.

Elle ne constitue pas une interprétation.

Elle explicite uniquement une relation pouvant être justifiée par les faits auxquels elle est reliée.

---

# Les faits précèdent toujours les relations

Une relation documentaire ne peut exister qu'entre des faits documentaires.

Elle ne dépend jamais directement d'une transcription ou d'une source primaire.

Les faits constituent les unités élémentaires à partir desquelles les relations sont établies.

---

# Les relations ne remplacent pas les faits

Créer une relation ne modifie jamais les faits documentaires.

Les faits demeurent indépendants.

La relation constitue une information supplémentaire décrivant leur organisation.

---

# Une relation est elle-même traçable

Toute relation documentaire demeure reliée aux faits qui la composent.

Par conséquent, elle reste indirectement reliée :

* aux séquences documentaires ;
* aux documents ;
* aux sources primaires.

La chaîne documentaire demeure complète.

---

# Nature des relations

Une relation documentaire peut exprimer, par exemple :

* une continuité ;
* une complémentarité ;
* une chronologie ;
* une cohérence ;
* une contradiction ;
* une précision ;
* une évolution.

Cette liste n'est pas limitative.

Les catégories de relations pourront évoluer avec le domaine documentaire.

---

# Les contradictions

Une contradiction constitue une relation documentaire à part entière.

Elle ne représente pas une erreur.

Elle décrit l'existence de faits incompatibles ou divergents.

InsouWiki documente ces contradictions sans chercher à les supprimer.

---

# Les connaissances

Les connaissances documentaires sont construites à partir de réseaux de faits documentaires reliés par des relations documentaires.

La connaissance ne remplace ni les faits ni les relations.

Elle en constitue l'organisation.

---

# Neutralité documentaire

Les relations documentaires sont établies selon les mêmes règles pour toutes les sources primaires.

Elles ne dépendent ni des opinions exprimées ni de l'identité des auteurs.

---

# Invariants

Toute relation documentaire respecte les règles suivantes :

* elle relie au moins deux faits documentaires ;
* elle demeure reliée aux faits qu'elle organise ;
* elle conserve la possibilité permanente de remonter jusqu'aux preuves documentaires ;
* elle ne contient aucune affirmation dépourvue de justification documentaire.

---

# Conclusion

Les relations documentaires constituent le niveau d'organisation situé entre les faits documentaires et les connaissances documentaires.

Elles permettent à InsouWiki de construire progressivement un patrimoine de connaissances tout en préservant la traçabilité complète des preuves qui le fondent.
