# MÉTHODE DE SÉQUENÇAGE

## Objet

Ce document définit les principes utilisés par InsouWiki pour transformer une transcription en un ensemble de séquences documentaires.

Il décrit une méthode documentaire.

Il ne dépend d'aucun modèle d'intelligence artificielle ni d'aucune technologie particulière.

---

# Principe

Une transcription constitue une représentation textuelle d'un document.

Elle n'est pas encore directement exploitable comme preuve documentaire.

Le rôle du séquençage est d'organiser cette transcription en unités documentaires cohérentes, horodatées et citables.

Ces unités sont appelées **séquences documentaires**.

---

# Objectif

Une séquence documentaire doit permettre au lecteur de comprendre une idée sans devoir consulter immédiatement le reste du document.

Elle constitue la plus petite unité documentaire pouvant être citée indépendamment tout en conservant son sens.

---

# Règles documentaires

Une séquence documentaire respecte les principes suivants.

## 1. Fidélité

Le texte d'une séquence est reproduit fidèlement.

InsouWiki ne reformule pas le contenu.

La séquence représente ce qui est effectivement présent dans la source.

---

## 2. Horodatage

Chaque séquence possède :

* un instant de début ;
* un instant de fin.

Ces informations permettent de retrouver immédiatement le passage correspondant dans la source primaire.

---

## 3. Cohérence

Une séquence regroupe un développement cohérent.

Elle ne doit pas interrompre une idée en cours de construction.

Inversement, deux idées distinctes doivent appartenir à deux séquences différentes.

---

## 4. Autonomie

Une séquence doit pouvoir être comprise indépendamment.

Le lecteur doit pouvoir la citer sans perdre le sens essentiel du passage.

---

## 5. Atomicité

Une séquence décrit une idée documentaire principale.

Lorsqu'un changement significatif de sujet ou d'argumentation apparaît, une nouvelle séquence est créée.

---

# Séquençage

Le découpage d'une transcription peut être proposé par un moteur d'intelligence artificielle ou par un algorithme.

Cependant, la validité d'une séquence est déterminée exclusivement par les règles documentaires définies dans ce document.

La méthode documentaire prime toujours sur la technologie employée.

---

# Vérifiabilité

Chaque séquence conserve un lien permanent avec :

* le document ;
* la transcription ;
* les instants correspondants dans la source.

Toute connaissance produite ultérieurement devra pouvoir être reliée à une ou plusieurs séquences documentaires.

---

# Évolution

La méthode de séquençage pourra évoluer afin d'améliorer la qualité du découpage documentaire.

Toute évolution devra cependant préserver les principes suivants :

* fidélité ;
* horodatage ;
* cohérence ;
* autonomie ;
* atomicité ;
* traçabilité.

Ces principes constituent le cadre permanent de la méthode documentaire d'InsouWiki.

---

# Conclusion

Le séquençage documentaire ne consiste pas à découper un texte.

Il consiste à identifier les unités élémentaires de preuve à partir desquelles une connaissance pourra être construite.

La qualité d'InsouWiki dépend directement de la qualité de ces séquences documentaires.
