# OBJET — EXPLORATION ITEM

## Objet

Ce document définit la notion d'**Exploration Item** au sein du domaine d'InsouWiki.

Un Exploration Item représente un élément présenté au lecteur au cours d'une exploration documentaire.

Il constitue l'unité fondamentale de l'expérience d'exploration.

---

# Pourquoi cet objet existe

Une exploration documentaire ne présente pas directement l'ensemble du patrimoine documentaire.

Elle sélectionne progressivement des éléments pertinents.

Ces éléments peuvent être de nature très différente.

Ils méritent néanmoins d'être considérés de manière homogène du point de vue de l'expérience du lecteur.

---

# Définition

Un Exploration Item est un élément documentaire intégré dans une exploration.

Il possède un sens pour le lecteur indépendamment de sa nature technique.

Il constitue une étape du parcours documentaire.

---

# Nature des Exploration Items

Un Exploration Item peut représenter notamment :

* une observation documentaire ;
* une connaissance documentaire ;
* un fait documentaire ;
* une séquence documentaire ;
* un document ;
* une citation ;
* une évolution documentaire ;
* une convergence documentaire ;
* une divergence documentaire.

Cette liste n'est pas limitative.

L'objet a vocation à accueillir toute information pertinente pour une exploration.

---

# Un Exploration Item n'est pas...

Un Exploration Item n'est pas :

* un nouveau type de document ;
* une copie du patrimoine documentaire ;
* une synthèse autonome ;
* une conclusion.

Il constitue simplement la représentation d'un élément du patrimoine dans le contexte d'une exploration.

---

# Un Exploration Item est...

Un Exploration Item est :

* contextualisé ;
* traçable ;
* vérifiable ;
* explicable ;
* relié aux preuves.

Il conserve toujours un lien vers son origine documentaire.

---

# Relation avec l'exploration documentaire

Une exploration documentaire est composée d'Exploration Items.

Ces éléments peuvent être organisés selon différents parcours.

Leur ordre dépend de l'intention du lecteur et du déroulement de l'exploration.

Ainsi, un même fait documentaire peut apparaître dans plusieurs explorations différentes sans être dupliqué.

---

# Le rôle de l'intelligence artificielle

L'intelligence artificielle peut sélectionner, ordonner ou rapprocher des Exploration Items.

Elle ne modifie jamais leur contenu documentaire.

Elle agit sur leur présentation au sein de l'exploration, jamais sur les preuves auxquelles ils demeurent reliés.

---

# Vision

Le patrimoine documentaire constitue la mémoire d'InsouWiki.

Les Exploration Items constituent la manière dont cette mémoire est présentée au lecteur.

Ils assurent le lien entre les objets du domaine et l'expérience documentaire.

Ainsi, l'exploration ne consiste pas à parcourir directement des structures techniques.

Elle consiste à rencontrer progressivement des éléments documentaires organisés de manière cohérente, toujours reliés aux preuves qui leur donnent leur valeur.
