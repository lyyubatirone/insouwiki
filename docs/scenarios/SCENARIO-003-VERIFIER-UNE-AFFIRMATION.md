# SCENARIO 003 — VÉRIFIER UNE AFFIRMATION

## Objet

Ce scénario illustre la manière dont InsouWiki permet à un lecteur de vérifier une affirmation à partir de preuves documentaires.

L'objectif n'est pas de confirmer ou d'infirmer immédiatement cette affirmation.

Il est de permettre au lecteur de retrouver les documents sur lesquels elle peut être évaluée.

---

# Situation

En parcourant un réseau social, Claire lit le message suivant :

> « Jean-Luc Mélenchon a déclaré vouloir repousser l'âge de départ à la retraite à 65 ans. »

Cette affirmation lui paraît surprenante.

Elle souhaite vérifier ce qui a réellement été dit.

Elle ouvre InsouWiki.

---

# L'intention du lecteur

Claire ne cherche pas un commentaire.

Elle souhaite répondre à une question simple :

> « Que montrent les documents concernant cette affirmation ? »

Cette intention devient le point de départ de son exploration.

---

# Identification du sujet

InsouWiki identifie les principaux éléments de la demande.

* Auteur : Jean-Luc Mélenchon
* Sujet documentaire : Retraites
* Élément à vérifier : âge de départ à la retraite

Le système prépare alors une exploration documentaire adaptée.

---

# Les observations documentaires

InsouWiki présente les observations documentaires pertinentes.

Par exemple :

* les principales affirmations documentées concernant ce sujet ;
* les éventuelles évolutions observables ;
* les continuités documentaires.

Aucune conclusion n'est affichée.

Le lecteur choisit librement les observations qu'il souhaite explorer.

---

# Les preuves

Claire sélectionne une observation.

InsouWiki lui présente progressivement :

* les connaissances documentaires associées ;
* les faits documentaires ;
* les séquences concernées ;
* les transcriptions ;
* les documents d'origine.

Chaque étape permet de mieux comprendre le contexte.

---

# Le contexte

Claire consulte la vidéo d'origine.

Elle écoute l'intervention complète.

Elle peut ainsi distinguer :

* les propos effectivement tenus ;
* leur contexte ;
* leur date ;
* leur place dans le raisonnement de l'auteur.

La vérification documentaire ne se limite jamais à une citation isolée.

---

# Une nouvelle exploration

Au cours de cette vérification, Claire découvre que d'autres documents abordent le même sujet.

Elle décide de poursuivre son exploration afin de mieux comprendre l'évolution des prises de position au fil du temps.

Une simple vérification devient ainsi une exploration documentaire plus large.

---

# Ce qu'InsouWiki a fait

InsouWiki a :

* identifié le sujet documentaire concerné ;
* retrouvé les documents pertinents ;
* organisé les preuves disponibles ;
* facilité la vérification de l'affirmation.

---

# Ce qu'InsouWiki n'a pas fait

InsouWiki n'a pas :

* décidé si le message du réseau social était volontairement trompeur ;
* attribué une intention à son auteur ;
* produit un verdict définitif ;
* remplacé l'analyse du lecteur.

Il a uniquement organisé les éléments documentaires permettant une vérification rigoureuse.

---

# Enseignement

Vérifier une affirmation ne consiste pas à rechercher une réponse toute faite.

Il s'agit de retrouver les documents qui permettent au lecteur de construire sa propre conclusion.

InsouWiki ne transforme pas une affirmation en vérité.

Il transforme une affirmation en une invitation à explorer les preuves disponibles.

Ainsi, le jugement demeure toujours celui du lecteur, éclairé par un patrimoine documentaire organisé, traçable et vérifiable.
