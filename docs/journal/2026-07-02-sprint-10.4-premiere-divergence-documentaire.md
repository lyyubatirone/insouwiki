# Journal — Sprint 10.4

## Première divergence documentaire

Le Sprint 10.4 achève la première génération des raisonneurs documentaires d'InsouWiki.

Après les évolutions, les continuités et les convergences documentaires, le projet apprend désormais à reconnaître les divergences documentaires.

Une divergence documentaire constitue l'observation que plusieurs auteurs expriment des affirmations différentes concernant un même sujet documentaire.

Comme pour les autres raisonnements documentaires, cette observation repose exclusivement sur des preuves documentaires identifiées.

La première implémentation adopte volontairement une règle simple.

Une divergence documentaire est reconnue lorsque plusieurs auteurs formulent des affirmations textuellement différentes concernant un même sujet documentaire.

Cette règle constitue une première étape.

Elle sera progressivement enrichie afin de distinguer les différences de formulation des divergences documentaires de fond, tout en conservant les exigences de prudence et de vérifiabilité qui fondent le projet.

L'observation produite est volontairement limitée :

> « Plusieurs auteurs expriment des affirmations différentes sur ce sujet dans les sources documentaires disponibles. »

Cette formulation décrit uniquement une différence observable entre plusieurs affirmations.

Elle ne permet pas de conclure à une opposition générale entre les auteurs concernés.

Le Sprint 10.4 complète ainsi la première bibliothèque des raisonnements documentaires d'InsouWiki.

Le projet est désormais capable d'observer :

* les évolutions documentaires ;
* les continuités documentaires ;
* les convergences documentaires ;
* les divergences documentaires.

Ces quatre raisonnements constituent les premières briques du langage documentaire d'InsouWiki.

Ils permettent de décrire un patrimoine documentaire selon plusieurs points de vue complémentaires, tout en respectant les principes de traçabilité, de prudence et de vérifiabilité qui fondent le projet.

Le Sprint 10.4 marque ainsi l'achèvement de la première génération des raisonneurs documentaires.
