# La transcription dans InsouWiki

## Préambule

La transcription constitue la première étape d'enrichissement d'un document audiovisuel.

Elle transforme un contenu sonore en une représentation textuelle exploitable tout en conservant un lien permanent avec le document d'origine.

La transcription ne remplace jamais le document.

Elle constitue une nouvelle ressource documentaire qui enrichit le patrimoine d'InsouWiki.

---

# Définition

Une transcription est une représentation textuelle d'un document audiovisuel.

Elle restitue le plus fidèlement possible le contenu oral d'un document primaire.

Elle est toujours associée à un document unique.

Une transcription n'existe jamais indépendamment du document dont elle est issue.

---

# Objectifs

La transcription poursuit plusieurs objectifs.

## Préserver

Transformer un contenu audiovisuel en un contenu textuel durable.

---

## Faciliter

Permettre la lecture d'un document sans nécessiter son écoute ou son visionnage.

---

## Préparer

Constituer la matière première des traitements documentaires suivants :

* segmentation ;
* résumé documentaire ;
* extraction des faits documentaires ;
* indexation ;
* recherche textuelle.

---

# Principes

Une transcription respecte les principes suivants.

## Fidélité

Elle cherche à restituer le contenu du document sans modifier le sens des propos.

---

## Traçabilité

Elle demeure liée en permanence au document d'origine.

---

## Neutralité

Elle ne contient aucune interprétation.

Elle reproduit le contenu du document.

---

## Évolutivité

Une transcription peut être remplacée par une version de meilleure qualité sans modifier le document auquel elle est associée.

Les différentes versions peuvent être conservées lorsque cela présente un intérêt documentaire.

---

# Informations minimales

Une transcription comporte au minimum :

* un identifiant permanent ;
* le document auquel elle appartient ;
* le texte transcrit ;
* la langue ;
* le moteur ou la méthode de transcription ;
* la date de création.

---

# Enrichissements futurs

Les versions futures pourront intégrer :

* les horodatages ;
* les locuteurs ;
* le niveau de confiance ;
* les annotations ;
* plusieurs langues ;
* plusieurs versions d'une même transcription.

Ces enrichissements ne remettent pas en cause la définition fondamentale de la transcription.

---

# Position dans le cycle documentaire

```text
Source
        │
        ▼
Point de diffusion
        │
        ▼
Document
        │
        ▼
Transcription
        │
        ▼
Segments
        │
        ▼
Faits documentaires
```

La transcription constitue la première couche de connaissance produite à partir d'un document.

Elle ouvre la voie à tous les enrichissements documentaires ultérieurs.

---

# Ce qu'une transcription n'est pas

Une transcription n'est pas :

* un résumé ;
* une interprétation ;
* une analyse ;
* une synthèse ;
* une traduction.

Elle représente uniquement le contenu textuel d'un document audiovisuel.

---

# Conclusion

La transcription est la première transformation documentaire réalisée par InsouWiki.

Elle ne crée pas de connaissance nouvelle.

Elle rend simplement le contenu d'un document exploitable par les traitements documentaires qui suivront.

Elle constitue ainsi le point de départ de la transformation progressive d'un document primaire en faits documentaires vérifiables.
