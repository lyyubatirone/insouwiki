# EXPERIENCES DOCUMENTAIRES

## Objet

Ce document décrit les principales expériences documentaires qu'InsouWiki doit rendre possibles.

Une expérience documentaire correspond à un parcours intellectuel permettant au lecteur de découvrir, comprendre, comparer ou vérifier un patrimoine documentaire.

Ces expériences sont indépendantes de toute interface.

Elles pourront être proposées par un site web, une application mobile, une API ou un assistant conversationnel.

---

# Principe fondamental

InsouWiki ne propose pas un parcours unique.

Il offre plusieurs façons d'explorer un même patrimoine documentaire.

Chaque expérience répond à une intention différente du lecteur.

Toutes reposent sur les mêmes principes :

* traçabilité ;
* vérifiabilité ;
* progressivité ;
* liberté d'exploration.

---

# Explorer un sujet documentaire

Le lecteur souhaite comprendre un sujet.

À partir du sujet documentaire, InsouWiki propose progressivement :

* les principales observations documentaires ;
* les connaissances documentaires associées ;
* les faits documentaires ;
* les preuves disponibles.

Cette expérience permet d'obtenir une vision d'ensemble avant d'approfondir certains aspects.

---

# Explorer un auteur

Le lecteur souhaite comprendre le parcours documentaire d'un auteur.

InsouWiki met en évidence notamment :

* les principaux sujets abordés ;
* les continuités documentaires ;
* les évolutions documentaires ;
* les documents associés.

Cette expérience privilégie une lecture chronologique et documentaire.

---

# Comparer plusieurs auteurs

Le lecteur souhaite comparer plusieurs auteurs concernant un même sujet documentaire.

InsouWiki met notamment en évidence :

* les convergences documentaires ;
* les divergences documentaires ;
* les preuves documentaires associées.

Cette comparaison repose exclusivement sur les documents disponibles.

---

# Vérifier une observation

Le lecteur souhaite comprendre pourquoi une observation documentaire a été produite.

À partir de cette observation, InsouWiki permet d'accéder progressivement :

* aux connaissances documentaires ;
* aux faits documentaires ;
* aux séquences concernées ;
* aux transcriptions ;
* aux documents d'origine.

Cette expérience garantit que chaque observation demeure entièrement vérifiable.

---

# Explorer librement

Une exploration documentaire ne suit pas nécessairement un parcours linéaire.

Le lecteur peut passer librement :

* d'un auteur à un sujet ;
* d'un sujet à un document ;
* d'une observation à une preuve ;
* d'une preuve à une autre observation.

Chaque découverte peut devenir le point de départ d'une nouvelle exploration.

---

# Le rôle des observations documentaires

Les observations documentaires constituent les principaux guides de l'exploration.

Elles attirent l'attention du lecteur sur ce qui est objectivement observable dans le patrimoine documentaire.

Elles n'imposent jamais une conclusion.

Elles proposent des pistes d'exploration.

---

# Le rôle de l'intelligence artificielle

L'intelligence artificielle facilite les différentes expériences documentaires.

Elle contribue à organiser le patrimoine documentaire, à identifier certaines relations et à produire des observations documentaires.

Elle ne décide jamais de ce que le lecteur doit penser.

Chaque expérience demeure fondée sur les preuves accessibles.

---

# Vision

Une expérience documentaire réussie ne consiste pas à fournir immédiatement une réponse.

Elle permet au lecteur de construire progressivement sa compréhension d'un sujet documentaire.

Chaque étape rapproche le lecteur des preuves.

Chaque preuve enrichit sa compréhension.

L'objectif d'InsouWiki est de transformer un patrimoine documentaire vaste et complexe en un environnement d'exploration où chacun peut découvrir, comparer, approfondir et vérifier selon ses propres questions et son propre rythme.
