# EXPLORATION DOCUMENTAIRE

## Objet

Ce document décrit la manière dont un lecteur explore un patrimoine documentaire au sein d'InsouWiki.

Il ne décrit ni une interface utilisateur, ni une implémentation technique.

Il définit le parcours intellectuel proposé au lecteur pour découvrir, comprendre et vérifier un sujet documentaire.

---

# Principe fondamental

Une exploration documentaire n'est pas une recherche de réponse.

C'est un parcours progressif au sein d'un patrimoine documentaire.

Chaque étape apporte de nouveaux éléments de compréhension tout en permettant au lecteur de revenir aux preuves qui les fondent.

InsouWiki accompagne cette exploration sans jamais s'y substituer.

---

# Les portes d'entrée

Une exploration documentaire peut commencer de différentes manières.

Le lecteur peut partir :

* d'un auteur ;
* d'un sujet documentaire ;
* d'un document ;
* d'une citation ;
* d'une question.

Ces différentes entrées conduisent progressivement vers le même patrimoine documentaire.

Aucune n'est privilégiée.

---

# Le rôle du sujet documentaire

Le sujet documentaire constitue le point de convergence de l'exploration.

Les documents, les faits, les connaissances et les observations documentaires s'y rattachent progressivement.

Le sujet documentaire permet d'organiser un patrimoine souvent très vaste sans imposer un parcours unique.

---

# Les observations documentaires

Les observations documentaires constituent le premier niveau de lecture proposé au lecteur.

Elles permettent de mettre rapidement en évidence ce qui est observable dans les preuves disponibles.

Les premières observations documentaires sont :

* les continuités ;
* les évolutions ;
* les convergences ;
* les divergences.

Ces observations ne remplacent jamais les preuves.

Elles orientent l'exploration.

---

# L'approfondissement

Chaque observation documentaire peut être développée.

Le lecteur peut accéder successivement :

* aux connaissances documentaires ;
* aux faits documentaires ;
* aux séquences documentaires ;
* aux transcriptions ;
* aux documents d'origine.

Chaque niveau apporte davantage de contexte.

Le lecteur reste libre d'interrompre son exploration à tout moment.

---

# Une exploration réversible

L'exploration documentaire est entièrement réversible.

Le lecteur peut partir :

* d'une observation pour retrouver les preuves ;
* d'un document pour découvrir les observations auxquelles il contribue.

Cette réversibilité constitue une propriété fondamentale d'InsouWiki.

Elle garantit que toute synthèse demeure reliée au patrimoine documentaire.

---

# Le rôle de l'intelligence artificielle

L'intelligence artificielle facilite l'exploration.

Elle aide notamment à :

* structurer le patrimoine documentaire ;
* identifier des faits documentaires ;
* produire certaines observations documentaires ;
* relier les informations entre elles.

Elle ne décide jamais du sens que le lecteur doit donner aux documents.

Le jugement demeure toujours humain.

---

# La liberté du lecteur

Chaque lecteur possède son propre parcours.

Certains souhaitent une synthèse rapide.

D'autres examinent chaque preuve.

L'architecture d'InsouWiki respecte ces différentes démarches.

Elle ne cherche jamais à imposer une conclusion.

Elle donne au lecteur les moyens de construire la sienne.

---

# Vision

Explorer un patrimoine documentaire ne consiste pas à obtenir une réponse immédiate.

Il s'agit de construire progressivement une compréhension fondée sur des preuves vérifiables.

InsouWiki ne remplace ni la lecture, ni l'analyse, ni l'esprit critique.

Il organise le patrimoine documentaire afin que chacun puisse l'explorer selon ses propres questions, son propre rythme et son propre niveau d'approfondissement.

L'objectif d'InsouWiki n'est pas de penser à la place du lecteur.

Il est de rendre possible une exploration documentaire plus riche, plus transparente et plus libre.
