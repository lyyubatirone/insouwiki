# OBJET — EXPLORATION DOCUMENTAIRE

## Objet

Ce document définit l'objet **Exploration documentaire** au sein du domaine d'InsouWiki.

L'exploration documentaire représente l'incarnation d'une intention de lecture.

Elle rassemble, dans un parcours cohérent, les différents éléments documentaires permettant au lecteur de comprendre progressivement un sujet.

---

# Pourquoi cet objet existe

Les documents, les faits, les connaissances et les observations existent indépendamment les uns des autres.

Le lecteur, en revanche, ne les consulte jamais de manière isolée.

Il réalise toujours une exploration.

Cette exploration constitue un objet du domaine à part entière.

Elle mérite donc d'être explicitement modélisée.

---

# Définition

Une exploration documentaire est un parcours documentaire organisé autour d'une intention.

Cette intention peut être :

* comprendre un sujet ;
* comparer plusieurs auteurs ;
* vérifier une affirmation ;
* retrouver le contexte d'une citation ;
* suivre l'évolution d'une idée ;
* découvrir un nouveau sujet.

Ces intentions correspondent aux scénarios de référence d'InsouWiki.

---

# Composition

Une exploration documentaire peut comprendre notamment :

* une intention d'exploration ;
* un ou plusieurs sujets documentaires ;
* des observations documentaires ;
* des connaissances documentaires ;
* des faits documentaires ;
* des séquences documentaires ;
* des documents d'origine.

Elle n'impose aucun parcours unique.

Elle organise simplement les éléments disponibles.

---

# Une exploration n'est pas...

Une exploration documentaire n'est pas :

* une réponse automatique ;
* un résumé ;
* une opinion ;
* une conclusion ;
* un raisonnement autonome.

Elle constitue un environnement permettant au lecteur d'exercer son propre jugement.

---

# Une exploration est...

Une exploration documentaire est :

* progressive ;
* réversible ;
* traçable ;
* vérifiable ;
* explicable ;
* ouverte.

Ces propriétés sont définies dans **QUALITÉS-D-UNE-EXPLORATION-DOCUMENTAIRE.md**.

---

# Cycle de vie

Une exploration documentaire commence lorsqu'un lecteur manifeste une intention.

Elle évolue au fil des découvertes documentaires.

Elle peut conduire vers d'autres explorations.

Une exploration ne possède pas nécessairement de fin.

Chaque découverte peut devenir le point de départ d'un nouveau parcours.

---

# Le rôle de l'intelligence artificielle

L'intelligence artificielle contribue à construire l'exploration.

Elle rapproche les documents.

Elle organise les observations.

Elle suggère des parcours documentaires.

Elle ne décide jamais à la place du lecteur.

---

# Vision

L'exploration documentaire constitue l'objet central de l'expérience proposée par InsouWiki.

Les documents représentent la mémoire du patrimoine documentaire.

Les observations mettent en évidence ce qui est documenté.

Les explorations permettent au lecteur de parcourir librement cet ensemble.

Ainsi, InsouWiki ne fournit pas des réponses.

Il construit des explorations documentaires fondées sur les preuves, afin que chacun puisse élaborer sa propre compréhension.
