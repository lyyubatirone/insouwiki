# DOMAINE — CONTINUITÉ DOCUMENTAIRE

## Objet

Ce document définit la notion de continuité documentaire au sein d'InsouWiki.

Une continuité documentaire constitue une observation produite à partir d'un ensemble de preuves documentaires.

Elle ne traduit ni une opinion, ni une interprétation.

Elle décrit uniquement une régularité observable dans le patrimoine documentaire.

---

# Définition

Une continuité documentaire est l'observation qu'une même affirmation apparaît de manière récurrente dans les sources documentaires disponibles.

Cette observation repose exclusivement sur les faits documentaires.

---

# Conditions documentaires

Une continuité documentaire peut être observée lorsque :

* les faits concernent le même auteur ;
* les faits concernent le même sujet documentaire ;
* les faits expriment la même affirmation ;
* plusieurs occurrences documentaires indépendantes sont disponibles.

---

# Principe de prudence

Une continuité documentaire ne permet jamais d'affirmer qu'une position est définitive ou permanente.

Elle indique uniquement que la même affirmation apparaît de manière récurrente dans les sources documentaires actuellement disponibles.

L'apparition ultérieure de nouvelles sources peut conduire à réviser cette observation.

---

# Traçabilité

Toute continuité documentaire doit permettre au lecteur de retrouver immédiatement les faits documentaires qui la justifient.

Aucune continuité documentaire ne peut exister sans preuves identifiées.

---

# Exemple

À partir de plusieurs faits documentaires contenant la même affirmation, InsouWiki pourra produire une observation telle que :

> « Cette affirmation apparaît de manière récurrente dans les sources documentaires disponibles. »

Cette formulation décrit exclusivement une observation documentaire.

Elle ne porte aucun jugement sur les intentions de l'auteur ni sur l'évolution future de sa position.

---

# Vision

La continuité documentaire complète naturellement l'évolution documentaire.

L'une décrit les changements observables.

L'autre décrit les permanences observables.

Toutes deux contribuent à rendre le patrimoine documentaire plus intelligible, tout en laissant au lecteur la liberté d'exercer son propre jugement.
