# OBJET — PORTE D'ENTRÉE DOCUMENTAIRE

## Objet

Ce document définit la notion de **Porte d'entrée documentaire** au sein du domaine d'InsouWiki.

Une porte d'entrée documentaire représente le point de départ d'une exploration.

Elle constitue le premier lien entre une intention du lecteur et le patrimoine documentaire.

---

# Pourquoi cet objet existe

Une exploration documentaire ne commence jamais directement par les documents.

Elle commence toujours par un élément qui attire l'attention du lecteur.

Cet élément devient une porte d'entrée vers le patrimoine documentaire.

Cette porte d'entrée mérite d'être considérée comme un objet du domaine.

---

# Définition

Une porte d'entrée documentaire est un élément à partir duquel une exploration peut être initiée.

Elle ne constitue pas une réponse.

Elle n'est pas non plus une conclusion.

Elle ouvre simplement un parcours documentaire.

---

# Exemples de portes d'entrée

Une exploration peut commencer à partir :

* d'un sujet documentaire ;
* d'un auteur ;
* d'une affirmation ;
* d'une citation ;
* d'une évolution documentaire ;
* d'une convergence documentaire ;
* d'une divergence documentaire ;
* d'un document particulier ;
* d'une séquence documentaire ;
* d'une découverte proposée par InsouWiki.

Cette liste n'est pas figée.

Elle pourra évoluer avec le patrimoine documentaire.

---

# Une porte d'entrée n'est pas...

Une porte d'entrée documentaire n'est pas :

* une recherche informatique ;
* une requête technique ;
* un résultat ;
* une synthèse ;
* une interprétation.

Elle constitue uniquement le commencement d'une exploration.

---

# Une porte d'entrée est...

Une porte d'entrée documentaire est :

* accessible ;
* explicite ;
* vérifiable ;
* ouverte ;
* orientée vers les preuves.

Elle ne remplace jamais l'exploration.

Elle la rend possible.

---

# Relation avec l'exploration documentaire

Une exploration documentaire possède toujours au moins une porte d'entrée.

Certaines explorations peuvent en posséder plusieurs.

Le lecteur peut ainsi commencer son parcours de différentes manières tout en convergeant progressivement vers les mêmes documents et les mêmes preuves.

La diversité des portes d'entrée enrichit l'expérience documentaire sans modifier le patrimoine documentaire lui-même.

---

# Le rôle de l'intelligence artificielle

L'intelligence artificielle peut suggérer des portes d'entrée pertinentes.

Elle peut rapprocher des sujets, mettre en évidence une citation ou proposer une découverte.

En revanche, elle ne décide jamais du parcours du lecteur.

Le choix de la porte d'entrée appartient toujours à celui qui explore.

---

# Vision

Une bibliothèque ne se résume pas à ses collections.

Elle se caractérise également par les nombreuses façons d'y entrer.

InsouWiki poursuit cette même ambition.

Chaque porte d'entrée constitue une invitation à explorer le patrimoine documentaire.

Quelle que soit celle choisie, le lecteur est progressivement conduit vers les documents, les preuves et la compréhension qu'il construira lui-même.
