# DOMAINE — SUJETS DOCUMENTAIRES

## Objet

Ce document décrit le concept de sujet documentaire au sein d'InsouWiki.

Il complète le modèle documentaire en introduisant une notion indispensable à l'organisation et à l'exploration du patrimoine documentaire.

---

# Constat

Les faits documentaires décrivent des affirmations observables dans les sources primaires.

Les relations documentaires permettent de relier certains faits.

Les connaissances documentaires organisent ces faits.

Cependant, les raisonnements documentaires supposent souvent de répondre à une question supplémentaire :

> **Ces faits parlent-ils de la même chose ?**

Cette notion ne peut pas être déduite uniquement des relations documentaires.

---

# Le sujet documentaire

Un sujet documentaire représente un thème abordé dans une ou plusieurs sources primaires.

Il ne constitue pas une opinion.

Il ne constitue pas une interprétation.

Il représente un point commun documentaire permettant d'organiser plusieurs faits.

Exemples :

* les retraites ;
* l'Union européenne ;
* la planification écologique ;
* la VIᵉ République.

---

# Rôle

Le sujet documentaire permet notamment :

* de regrouper des faits portant sur un même thème ;
* de suivre l'évolution d'une position dans le temps ;
* de comparer plusieurs auteurs sur un même sujet ;
* d'explorer le patrimoine documentaire par thème.

---

# Relations avec les faits

Un fait documentaire peut être associé à plusieurs sujets documentaires.

Inversement, un sujet documentaire peut regrouper un grand nombre de faits documentaires.

La relation est donc de type plusieurs-à-plusieurs.

Cette organisation reflète la richesse des sources primaires.

---

# Les sujets ne remplacent pas les faits

Le sujet documentaire ne constitue jamais une preuve.

Les preuves demeurent exclusivement les faits documentaires et les séquences documentaires qui les fondent.

Le sujet documentaire constitue uniquement une structure d'organisation du patrimoine documentaire.

---

# Construction

Dans les premières versions d'InsouWiki, les sujets documentaires pourront être proposés par l'intelligence artificielle.

Toute proposition devra rester reconstructible à partir des faits documentaires concernés.

Les méthodes d'identification des sujets pourront évoluer sans remettre en cause le reste du domaine.

---

# Utilisation

Les sujets documentaires pourront être utilisés par de nombreux services :

* recherche documentaire ;
* construction des connaissances ;
* détection des évolutions ;
* comparaison d'auteurs ;
* chronologies documentaires ;
* navigation dans le patrimoine documentaire.

Ils constituent un point de convergence entre plusieurs raisonnements documentaires.

---

# Principe de prudence

L'identification d'un sujet documentaire constitue une proposition documentaire.

Elle ne doit jamais être présentée comme une vérité absolue.

Le lecteur conserve toujours la possibilité d'explorer directement les faits documentaires et leurs preuves.

---

# Vision

Les sujets documentaires permettent de structurer progressivement le patrimoine documentaire sans modifier les preuves qui le composent.

Ils facilitent l'exploration des sources primaires tout en respectant le principe fondamental d'InsouWiki :

la preuve documentaire précède toujours l'organisation documentaire.
