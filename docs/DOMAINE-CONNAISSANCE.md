# DOMAINE — CONNAISSANCE DOCUMENTAIRE

## Objet

Ce document décrit le modèle documentaire utilisé par InsouWiki pour transformer progressivement une source primaire en une connaissance vérifiable.

Il complète le document `DOMAINE.md` en introduisant les concepts propres à l'enrichissement documentaire.

---

# Principe fondamental

Une connaissance ne peut exister sans preuve.

Dans InsouWiki, toute connaissance est rattachée à une ou plusieurs séquences documentaires issues d'une source primaire.

La preuve documentaire précède toujours l'interprétation.

---

# La chaîne documentaire

Chaque transformation enrichit le patrimoine documentaire sans remplacer les informations produites précédemment.

```text
Source primaire
        │
        ▼
Document
        │
        ▼
Transcription
        │
        ▼
Séquences documentaires
        │
        ▼
Faits documentaires
        │
        ▼
Connaissances reliées
```

Chaque niveau conserve un lien explicite avec le précédent.

La traçabilité est ainsi garantie jusqu'à la source originale.

---

# La séquence documentaire

La séquence documentaire constitue l'unité fondamentale de connaissance d'InsouWiki.

Une séquence représente une portion continue d'un document.

Elle possède notamment :

* un document d'origine ;
* un instant de début ;
* un instant de fin ;
* un contenu textuel ;
* un identifiant permanent.

Une séquence ne contient aucune interprétation.

Elle décrit uniquement ce qui est effectivement présent dans la source.

---

# Horodatage

Chaque séquence est horodatée.

L'horodatage permet :

* de retrouver immédiatement le passage correspondant dans la source originale ;
* de citer précisément une affirmation ;
* de vérifier une information sans ambiguïté ;
* de partager un lien pointant directement vers le moment concerné.

L'horodatage constitue un élément essentiel de la preuve documentaire.

---

# Les faits documentaires

Un fait documentaire est une information produite à partir d'une ou plusieurs séquences documentaires.

Un fait ne remplace jamais les séquences dont il est issu.

Il conserve en permanence la liste des preuves documentaires qui le justifient.

Ainsi, chaque fait peut être vérifié indépendamment.

---

# Les connaissances

Une connaissance est un ensemble organisé de faits documentaires reliés entre eux.

Elle constitue le niveau le plus élaboré du patrimoine documentaire.

Toute connaissance demeure traçable jusqu'aux séquences documentaires qui la fondent.

---

# Principe de vérifiabilité

À tout moment, un lecteur doit pouvoir parcourir le chemin inverse :

Connaissance → Fait documentaire → Séquence documentaire → Transcription → Document → Source primaire.

Ce parcours constitue le principe fondamental de vérifiabilité d'InsouWiki.

---

# Vision

InsouWiki ne cherche pas à remplacer le jugement du lecteur.

Il organise les preuves documentaires afin que chacun puisse comprendre, vérifier et exercer librement son esprit critique.

La confiance ne repose jamais sur l'intelligence artificielle.

Elle repose sur la possibilité permanente de revenir à la source primaire.
