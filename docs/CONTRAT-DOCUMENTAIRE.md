# Contrat documentaire des enrichissements

## Préambule

InsouWiki enrichit progressivement un patrimoine documentaire constitué de sources primaires.

Chaque enrichissement documentaire ajoute une nouvelle couche de connaissance sans jamais modifier ni remplacer le document d'origine.

Le présent contrat définit les garanties que tout enrichissement documentaire doit respecter.

Ces principes s'appliquent à la transcription, à la segmentation, aux faits documentaires ainsi qu'à tout enrichissement futur.

---

# 1. Principe de conservation

Le document primaire constitue la référence documentaire.

Aucun enrichissement ne modifie son contenu.

Le document original est conservé intégralement.

---

# 2. Principe de traçabilité

Tout enrichissement demeure relié au document dont il est issu.

Lorsque cela est possible, il est également relié à un passage précis de ce document.

La preuve documentaire reste accessible à tout moment.

---

# 3. Principe de neutralité

Les enrichissements documentaires décrivent les documents.

Ils ne modifient ni le sens ni le contenu des informations produites par la source.

---

# 4. Principe de transparence

Chaque enrichissement indique son origine.

Lorsque celui-ci est produit automatiquement, les informations suivantes sont conservées :

* moteur utilisé ;
* version du moteur ;
* date de production ;
* paramètres significatifs lorsque cela est pertinent.

---

# 5. Principe de reproductibilité

Un enrichissement doit pouvoir être reproduit dans des conditions comparables.

La méthode de production doit rester documentée.

---

# 6. Principe de versionnement

Un enrichissement peut être amélioré au cours du temps.

Une nouvelle version ne remet pas en cause l'existence des versions précédentes lorsqu'elles présentent un intérêt documentaire.

---

# 7. Principe d'évolutivité

Chaque enrichissement constitue une nouvelle couche documentaire.

De nouvelles couches pourront être ajoutées sans remettre en cause les précédentes.

---

# Les enrichissements documentaires

Le modèle documentaire d'InsouWiki prévoit notamment les enrichissements suivants :

* Transcription
* Segment
* Fait documentaire

D'autres enrichissements pourront être introduits ultérieurement tout en respectant le présent contrat.

---

# Conclusion

Le patrimoine documentaire constitue la mémoire d'InsouWiki.

Les enrichissements documentaires constituent les couches de connaissance qui permettent au lecteur d'explorer ce patrimoine.

Aucun enrichissement n'a vocation à remplacer le document primaire.

Tous contribuent à en faciliter la compréhension, tout en conservant un lien permanent avec la preuve documentaire originale.
